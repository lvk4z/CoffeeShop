A small Django project providing a REST API backend and a frontend app for a coffee shop.

- Backend: Django project located at [`coffee_shop/manage.py`](coffee_shop/manage.py) with main settings in [`coffee_shop.settings`](coffee_shop/coffee_shop/settings.py).
- API app: code and endpoints under [api/](api/) (see [`api/urls.py`](api/urls.py) and [api/tests.py](api/tests.py)).
- Frontend: single-page app in [frontend/](frontend/) (see [`frontend/package.json`](frontend/package.json) and [`frontend/webpack.config.js`](frontend/webpack.config.js)).
- Requirements: [`coffee_shop/requirements.txt`](coffee_shop/requirements.txt)
- Procfile: [`coffee_shop/Procfile`](coffee_shop/Procfile)
- Media uploads served from the [media/](media/) directory.
